# Data Types
---

- #### Text Type:
  - str
  ```python
  x = str("Hello World")
  ```
- #### Numeric Types:
  - int
  ```python
  x = int(20)
  ```
  - float
  ```python
  x = float(20.5)
  ```
  - complex
  ```python
  x = complex(1j)
  ```
- #### Sequence Types:
  - list
  ```python
  x = list(("apple", "banana", "cherry"))
  ```
  - tuple
  ```python
  x = tuple(("apple", "banana", "cherry"))
  ```
  - range
  ```python
  x = range(6)
  ```
- #### Mapping Type:
  - dict
  ```python
  x = dict(name="John", age=36)
  ```
- #### Set Types:
  - set
  ```python
  x = set(("apple", "banana", "cherry"))
  ```
  - frozenset
  ```python
  x = frozenset(("apple", "banana", "cherry"))
  ```
- #### Boolean Type:
  - bool
  ```python
  x = bool(5)
  ```
- #### Binary Types:
  - bytes
  ```python
  x = bytes(5)
  ```
  - bytearray
  ```python
  x = bytearray(5)
  ```
  - memoryview
- ```python
  x = memoryview(bytes(5))
  ```
