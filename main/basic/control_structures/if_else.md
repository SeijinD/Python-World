# If...Else
---

- ### If
~~~~
if test expression:
    statement(s)
~~~~
- ### If...else
~~~~
if test expression:
    Body of if
else:
    Body of else
~~~~
- ### If...elif...else
~~~~
if test expression:
    Body of if
elif test expression:
    Body of elif
else: 
    Body of else
~~~~
